# Learn to Code with Robots

Welcome! This is a hands-on course that teaches you to **code from scratch** by
building battle robots. You don't need to know anything about programming. We'll
explain every idea in plain words, with examples you can try right away.

Each lesson is short and builds on the one before it. By the end you'll be able to use
**every** feature RobocodeJs offers, and you'll have learned the core ideas that all
programmers use.

## How it works

- Read the lesson, then **paste the code into your own bot** and press **Deploy** to watch
  it run in the arena on the right.
- Try the **Experiment** at the end of each lesson: that's where the learning sticks.
- Stuck? Each lesson has a **Common questions** section for the usual surprises.

> **First time here?** Lesson 1 shows you how to make your very first bot. Start there!

## The lessons

**Part 1: Getting moving**

1. [Hello, bot!](/learn/hello): make a bot and give it a name
2. [Do something every moment](/learn/events): events and the game clock
3. [Move!](/learn/move): driving around the arena
4. [Which way? The compass](/learn/turn): turning and directions

**Part 2: Fighting back**

5. [Bumping into walls](/learn/walls): making decisions with `if`
6. [Fire the turret](/learn/fire): shooting (when you're ready)
7. [See your enemies: the radar](/learn/radar): scanning and lists
8. [Take aim](/learn/aim): pointing at a target

**Part 3: Thinking bots**

9. [Good things take time](/learn/waiting): waiting for actions to finish
10. [Remembering things](/learn/state): giving your bot a "brain" with states
11. [Survival](/learn/survival): health, getting hit, and running away
12. [Rhythms and timers](/learn/timers): doing things on a schedule

**Part 4: Going pro**

13. [Maps and math](/learn/navigation): navigating with markers and geometry
14. [Leading a moving target](/learn/leading): hitting movers by aiming ahead of them
15. [Teamwork and graduation](/learn/teamwork): talking to teammates + your final challenge
