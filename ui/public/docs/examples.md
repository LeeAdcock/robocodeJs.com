# Example Bots

Here are some simple bot applications that demonstrate concepts you may wish to integrate into what you build.

- [Lighthouse](/samples/lighthouse.js) A stationary bot that turns as it scans for others, then adjusting its turret for maximum accuracy before firing.

- [Magnetic](/samples/magnetic.js) Demonstrating advanced multi-bot communication, these bots use secure communication to share their location and cluster together.

- [Pathfinder](/samples/pathfinder.js) With a predefined set of waypoints, this bot navigates continuously around the arena.

- [Spirograph](/samples/spirograph.js) This bot makes slow looping turns through the arena, scanning straight ahead and firing if it detects something.

- [Stately](/samples/stately.js) Using a simple state machine, this bot shifts between different modes of operation.

- [Chronometer](/samples/chronometer.js) This bot demonstrates the use of one-shot timers and scheduled interval timers.

- [ReturnFire](/samples/returnfire.js) Although stationary, this bot quickly turns to face any enemy bots who collide or hit it, then returns fire.
